## ModuleNotFoundError

problems:

solution:

python interpreter add packages
这困扰了我一天的时间，一直以为是 python 环境的问题, shell 下命令行装了好多遍。后来发现 PyCharm 配置的 inertpreter ，相关文件全在 venv 文件夹下，添加的 package 都配置文件到 venv 文件下，所谓的相关开发环境就是 venv 内的相关文件，日后只需配置此即可。
